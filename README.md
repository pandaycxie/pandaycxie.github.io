# Yichuan Xie

This repository hosts the academic homepage for Yichuan Xie at
`https://pandaycxie.github.io`.

The site is intentionally lightweight: plain `index.html`, `style.css`, and a
linked PDF CV. GitHub Pages can serve it directly from the repository root
without a build step.

## Maintenance

- Edit `index.html` for homepage content.
- Edit `style.css` for visual styling.
- Replace `Yichuan_Xie_CV_2026.pdf` when updating the full CV.
